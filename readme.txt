[RayPay](https://raypay.ir) payment method for [myCRED](https://wordpress.org/plugins/mycred/).

== Description ==

[RayPay](https://raypay.ir) is one of the Financial Technology providers in Iran. One of the services which RayPay provides is payment gateway service. This plugin enables myCRED to use RayPay as a payment gateway.

== Installation ==

A complete documentation for installing this plugin is available [here](https://raypay.ir/Plugins).
